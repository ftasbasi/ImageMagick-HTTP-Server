# ImageMagick-HTTP-Server
A simple image editing HTTP Server using Magick++ API

Compile:
c++ `Magick++-config --cxxflags --cppflags` -O2 -o server server.cpp  `Magick++-config --ldflags --libs`

Run: (Linux terminal)
./server
